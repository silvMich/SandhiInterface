## EEGLab
For the analysis and visualization of the .xdf file we suggest the use of the Add-On [EEGLab](https://eeglab.org/others/How_to_download_EEGLAB.html) in Matlab.

It is better to install manually the add-on since it doesn't support .xdf files by default and it is mandatory to install a [plug in](https://github.com/xdf-modules/xdf-EEGLAB).
